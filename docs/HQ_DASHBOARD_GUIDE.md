---

## 📊 본사 담당자 대시보드 개요

협력사 관리에 특화된 대시보드로, 금형 위치 추적, 관리 표준 마스터 설정, 협력사별 성과 모니터링 등의 기능을 제공합니다.

### 관리자 대시보드와의 차이점

| 기능 | 관리자 | 담당자 |
|------|--------|--------|
| 시스템 전체 조회 | ✅ | ✅ |
| 사용자 관리 | ✅ | ❌ |
| 협력사 관리 | ✅ | ✅ |
| 금형 위치 추적 | ✅ | ✅ |
| 관리 표준 마스터 | ✅ | ✅ |
| 승인 처리 | ✅ | ✅ |
| 시스템 설정 | ✅ | ❌ |

### 주요 기능

1. **협력사 관리**
   - 협력사별 금형 현황 (50개 협력사)
   - 성과 모니터링 및 평가
   - 계약 관리
   - 평가 등급 (A+, A, B, C, D)

2. **금형 위치 추적**
   - 협력사별 금형 위치 (1,234개)
   - 이동 이력 추적
   - 위치 기반 알림

3. **관리 표준 마스터**
   - 점검 표준 설정 (일상/정기/습합/세척)
   - 체크리스트 템플릿 (8개 카테고리)
   - 문서 양식 관리
   - 배포 및 버전 관리

4. **승인 및 검토**
   - 수리 요청 승인
   - 이관 요청 승인
   - 점검 결과 검토
   - 예외 사항 처리

## 🏢 협력사 관리

### 협력사 평가 시스템

#### 평가 항목 (100점 만점)

1. **점검 완료율** (20점)
   - 목표: 95% 이상
   - 측정: 완료된 점검 / 예정된 점검

2. **평균 응답 시간** (20점)
   - 목표: 3시간 이내
   - 측정: 요청부터 응답까지 평균 시간

3. **고장률** (20점)
   - 목표: 2% 이하
   - 측정: 고장 건수 / 전체 금형 수

4. **가동률** (20점)
   - 목표: 85% 이상
   - 측정: 가동 중인 금형 / 전체 금형

5. **품질 점수** (20점)
   - 목표: 90점 이상
   - 측정: 점검 결과 평균 점수

#### 등급 기준

| 등급 | 점수 | 설명 |
|------|------|------|
| A+ | 95-100 | 최우수 |
| A | 90-94 | 우수 |
| B | 80-89 | 양호 |
| C | 70-79 | 보통 |
| D | 70 미만 | 개선 필요 |

## ✅ 승인 및 검토

### 승인 유형

#### 1. 수리 요청 승인
- 긴급도: 긴급/높음/보통/낮음
- 예상 비용 검토
- 예상 기간 확인
- 수리 업체 선정

#### 2. 이관 요청 승인
- 이관 사유 확인
- 목적지 확인
- 운송 방법 검토
- 체크리스트 확인

#### 3. 점검 결과 검토
- 점검 완료 여부
- 발견사항 확인
- 조치사항 검토
- 후속 조치 필요 여부
